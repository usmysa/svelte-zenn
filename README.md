## Technology

- Framework
  - [Next.js](https://github.com/vercel/next.js)
  - [SvelteKit](https://github.com/sveltejs/kit)
- Monorepo tool
  - [Turborepo](https://github.com/vercel/turbo)
- JavaScript Runtime
  - [Bun](https://github.com/oven-sh/bun)
- Linter, Formatter
  - [Biome](https://biomejs.dev/)

## Quick Start

### Install dependencies

```sh
bun install
```

### Run locally

```sh
bun dev
```

- API
  - http://localhost:3000/
- WEB
  - http://localhost:5173/
